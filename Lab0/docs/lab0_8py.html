Detailed Description

This file contains code which uses PWM output with varying duty cycle to change the brightness of an LED.

The LED slowly reaches its max brightness over a 5 second time period and then resets and continues this cycle indefinitely.

def lab0.main ( )

Call set up channel and then set LED brightness This function first initializes the board by calling the led_setup function, then it iterares from 0 to 100 in an indefinite loop setting the LED brightness percent to the iterated value.

It delays by 50ms between setting values so that the LED brightness reaches its max over a 5 second time period. It resets the LED value after reaching the max and then repeats.
